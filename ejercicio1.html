HTML, son las siglas de (Hypertext Markup Languaje), que puuede traducirse como lenguaje de marcas o marcador de hipertexto.

El lenguaje HTML se emplea para crear paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta de cambiar de un navegador a otro y tambien puede cambiarse de version a otra de un mismo navegador,
pero suele tener un nombre similar.

El lenjuaje HTML sirve para describir la estructura basica de una pagiina y organizar la forma en el que se mostrara su contenido, ademas que HTML permite incluir enlaces hacia otras paginas o documentos.

HTML se compone de etiquetas que se escriben entre los simbolos de menor que y mayor que.
